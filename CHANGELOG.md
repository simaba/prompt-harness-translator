# Changelog

## [0.1.0] - 2026-06-25

### Added

- YAML-frontmatter parsing for the supported Markdown source shape.
- Validation for frontmatter structure and core fields.
- Codex, Cursor, and generic output paths.
- Preserved unmapped metadata and explicit lossy-conversion warnings.
- Regression tests for nested, multiline, invalid, and unsupported inputs.
- Supported-format documentation and public-release guidance.

### Scope

This is a starter translation utility. It is not a full interoperability standard and does not guarantee semantic or runtime equivalence across prompt and agent ecosystems.
