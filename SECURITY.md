# Security

This repository is intended for workflow automation, evaluation, and documentation.

## Guardrails

- Never commit real credentials, customer data, vehicle logs, or regulated safety evidence.
- Use synthetic or sanitized examples by default.
- Treat generated outputs as drafts that require human review.
- For safety, compliance, and legal content, keep a human sign-off step in the process.

## Reporting

Open a private security report with a minimal reproducible description of the issue.
